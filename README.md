- Memahami konsep dasar pemrograman PHP

- Mengerti best practices dalam PHP development

## 🚀 Cara Memulai

1. **Clone atau Download Repository**
   ```bash
   git clone https://github.com/mahendartea/php-mini-course.git
   ```

2. **Setup Environment**
   - Install XAMPP/Laragon/MAMP
   - Pastikan Apache dan PHP sudah running
   - Letakkan folder project di dalam `htdocs` (XAMPP) atau `www` (Laragon)

3. **Mulai Belajar**
   - Buka `pertemuan-01/README.md` untuk memulai
   - Ikuti urutan pertemuan secara berurutan
   - Praktikkan semua contoh kode

4. **Jalankan File PHP**
   ```
   http://localhost/php-mini-course/pertemuan-01/hello-world.php
   ```
